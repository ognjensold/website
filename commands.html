Commands

Here we will address every single command available

This documentation will provide you with the appropriate knowledge to expand your business

/start

This command will let you register to the system and start up your business

The system will ask you for an in-app captcha verification, after which you will provide some details to distinguish your store from others

/search

This command will let you search for any item/service registered on the system

The system will ask you for some filters like whether the item you are looking for is an online item, clothing or something else

/sell

This command will let you post a sale offer on the system

The system will ask you for some information about the product, price, shipping and payment method to make your product more reachable

/trade

This command will let you post a trade offer on the system

The system will ask you for some information about the items you wish to trade, the items you wish to get, shipping and transfer to make your trade more reachable

/service

This command will let you post a service offer on the system

The system will ask you for some infromation about your service, whether it is in person or online, pricing and payment methods

/connect

This command will let you connect to a user on the system

The system will ask you for the ID of the user, after which you will be able to see all of their posts and other information

/record

This command will let you record a transaction on the system

The system will ask you for some infromation about the transaction like quality, stars out of five and review. These will boost your store ranking and help you gain more connections

/report

This command will let you report a user

The system will ask you for some infromation about the user who you wish to report like their ID and proof attachments. Abusing this command will result in a permanent ban.
